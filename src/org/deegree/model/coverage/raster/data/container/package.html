This package contains the container classes which hide the actual loading of pixel data from the rasterdata.

An appropriate data container is loaded using the ServiceLoading mechanism, the default loading policy is memory loading, which directly loads RasterData directly into memory. Other implementations are (see d3_dataaccess/src/META-INF/):

  • org.deegree.dataaccess.raster.CachedRasterDataContainer
  • org.deegree.dataaccess.raster.LazyRasterDataContainer
