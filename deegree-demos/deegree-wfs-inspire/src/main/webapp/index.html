Welcome to the deegree 3 INSPIRE WFS-Demo preview. It showcases a fully transactional WFS for all INSPIRE Annex I Data Themes using PostGIS as backend.


Database setup
--------------

A PostgreSQL (8.0+) / PostGIS (1.0+) installation is required for this demo. You will have to create a spatially enabled PostgreSQL database first. Then, adapt the file WEB-INF/conf/jdbc/postgis.xml to match your database connection.

Run this SQL script on the database to create the necessary tables.


Testing the WFS
---------------

After database setup, you will have to restart the servlet container. You may then:

- Send requests (including transactions)
- Visit the status page
