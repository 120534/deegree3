deegree 3 GeoSciML demo (technology preview)
Welcome to the deegree 3 GeoSciML demo (technology preview).

Features

- Configuration uses the unmodified GeoSciML 2.0 schemas (GeoSciML 2.1 / 3.0 also possible, see Notes below)
- [Generic client] that allows to query GeoSciML features using WFS requests.
- Includes the deegree 3 WMS for styling/rendering of the unmodified (complex) GeoSciML features.
- See general technical features of the [deegree 3 WFS].

Live map (includes two layers rendered from complex GeoSciML features)

Notes

- In the online version, transactions are disabled, see the [deegree 3 WFS documentation] for setting up a local version.
- The demo styling is very simple, but sophisticated styling is supported as the [deegree 3 WMS] has extensive support for SLD + SE.
- If you can provide example datasets for GeoSciML 2.1 / 3.0, please contact the [deegree mailing lists], so we can update the demo.
